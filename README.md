### What I'm Building

**[ProTeach](https://proteachhomelearning.com)** — A full-stack homeschool learning platform for families across all 50 US states.

| Feature | Stack |
|---|---|
| Personalized weekly lesson plans (state DOE-aligned) | Next.js · TypeScript · Firebase |
| Educational game engine with XP, badges & audio | Framer Motion · Web Audio API |
| AI-powered content generation | Google Gemini AI |
| Subscription & billing management | Stripe |
| Parent, Student & Admin portals | Tailwind CSS · shadcn/ui |

**Power Liens Marketing Portal** — Internal operations portal for managing marketing deliverables, HubSpot CRM sync, and provider workflows.
